Diabetes Progression Prediction

Predicting diabetes progression using Machine Learning

Project Overview

This project predicts diabetes progression in patients using clinical features and machine learning models including Linear Regression, Ridge Regression, and Random Forest.

Dataset

  • Source: scikit-learn load_diabetes()
  • Samples: 442
  • Features: age, sex, BMI, blood pressure, s1-s6
  • Target: Disease progression metric after one year

Models and Results

Model R² Score
Linear Regression 0.516
Ridge Regression (alpha=0.001) 0.517
Random Forest Regressor (max_depth=10, n_estimators=500) 0.492

Code Snippet


from sklearn.datasets import load_diabetes
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression, Ridge
from sklearn.ensemble import RandomForestRegressor

# Load dataset
data = load_diabetes()
X, y = data.data, data.target

# Split dataset
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=55)

# Linear Regression
lr = LinearRegression()
lr.fit(X_train, y_train)
print("Linear R²:", lr.score(X_test, y_test))

# Ridge Regression
ridge = Ridge(alpha=0.001)
ridge.fit(X_train, y_train)
print("Ridge R²:", ridge.score(X_test, y_test))

# Random Forest Regressor
rf = RandomForestRegressor(max_depth=10, n_estimators=500, random_state=55)
rf.fit(X_train, y_train)
print("Random Forest R²:", rf.score(X_test, y_test))
