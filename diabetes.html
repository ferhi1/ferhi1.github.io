Diabetes Progression Prediction

Project Overview

This project aims to predict diabetes progression in patients using machine learning models based on clinical measurements. The goal is to estimate disease progression one year after baseline assessment.

Dataset

The project uses the Diabetes dataset from scikit-learn. This dataset contains information collected from 442 patients, with the following details:

  • Features: Age, Sex, BMI, Blood Pressure, and six blood serum measurements (s1 to s6)
  • Target: Quantitative measure of disease progression one year after baseline
  • Number of samples: 442

Models Used

Three machine learning models were implemented to predict diabetes progression:

  • Linear Regression: Basic linear approach for continuous prediction.
  • Ridge Regression: Linear model with L2 regularization (alpha=0.001) to reduce overfitting.
  • RandomForestRegressor: Ensemble model (n_estimators=500, max_depth=10) to capture non-linear relationships and interactions.

Results

The performance of the models was measured using the R² score on the test set:

  • Linear Regression: R² = 0.516
  • Ridge Regression: R² = 0.517
  • RandomForest Regressor: R² = 0.534

The RandomForest model achieved the best performance, indicating that ensemble methods can better capture the complex patterns in the dataset.

Python Code Example


from sklearn.datasets import load_diabetes
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression, Ridge
from sklearn.ensemble import RandomForestRegressor

# Load dataset
diabetes = load_diabetes()
X = diabetes.data
y = diabetes.target

# Split data
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=55)

# Linear Regression
lr = LinearRegression()
lr.fit(X_train, y_train)
print("Linear Regression R²:", lr.score(X_test, y_test))

# Ridge Regression
ridge = Ridge(alpha=0.001)
ridge.fit(X_train, y_train)
print("Ridge Regression R²:", ridge.score(X_test, y_test))

# RandomForest
rf = RandomForestRegressor(n_estimators=500, max_depth=10, random_state=55)
rf.fit(X_train, y_train)
print("RandomForest R²:", rf.score(X_test, y_test))
